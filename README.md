# jobRentAnalysis
## 岗位和租房联动分析系统
这个系统是用python的django框架和vue构建的一个岗位和租房联动分析系统，可以实时获取深圳每个区的岗位和房源信息，然后计算出两者之间的距离，取距离岗位最短的前几个房源信息进行一个分析，还有导航信息，可以提示如何到达公司。

请看前端GitHub仓库（https://github.com/zhongxiaoting/rent-house-web）
